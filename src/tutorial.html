Getting Started

Welcome to the Calculator! This simple tool allows you to perform basic mathematical operations with two numbers.

How to Use

  1. Enter the first number in the "First number" input field
  2. Enter the second number in the "Second number" input field
  3. Click one of the operation buttons to perform the calculation:
    • Add (+) - Adds the two numbers together
    • Subtract (-) - Subtracts the second number from the first
    • Multiply (√ó) - Multiplies the two numbers
    • Divide (√∑) - Divides the first number by the second
  4. The result will appear in the result section below the buttons
